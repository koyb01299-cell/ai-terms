# AI 핵심 용어 발음 사전 (학습 앱)

AI 비즈니스 필수 용어 162개를 학습하는 모바일 우선 웹앱입니다.
홈 화면에 추가하면 네이티브 앱처럼 전체화면으로 실행됩니다(PWA).

## 기능
- 홈 대시보드: 진도 링, 연속 학습(스트릭), 오늘 목표, 분류별 진도
- 학습: 플래시카드(뒤집기·스와이프), 4지선다 퀴즈(맞히면 자동 외움 처리)
- 사전: 분류·검색으로 전체 용어 탐색, 발음 듣기, 외움 체크
- 통계: 전체/분류별 암기 현황
- 발음: 브라우저 음성엔진(Web Speech API)

## 폴더 구조
```
ai-terms/
├── index.html         # 화면 구조
├── manifest.json      # PWA 매니페스트(앱 이름·아이콘)
├── icon-180.png       # iOS 홈화면 아이콘
├── icon-192.png / icon-512.png / icon-512-maskable.png  # Android/PWA 아이콘
├── css/styles.css
└── js/
    ├── data.js        # 용어 데이터(여기서 단어 추가/수정)
    └── app.js         # 앱 로직
```

## 홈 화면 추가 (iPhone Safari)
1. Vercel 등으로 배포해 URL 접속
2. Safari 공유 버튼 → "홈 화면에 추가"
3. "AI 용어" 아이콘이 생기고, 탭하면 전체화면 앱으로 실행

## 단어 추가
`js/data.js` 의 `TERMS` 에 `["한글","English","설명.","카테고리키"]` 추가.
